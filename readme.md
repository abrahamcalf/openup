> Open remote repositories from your terminal

## Install

```bash
$ npm install -g openup
```

or inside a repo just 

```bash
$ npx openup
```

## CLI

```console
Open remote repositories from your terminal

Usage:

  $ openup

Flags:

  -h, --help        Show help message and close
  -v, --version     View package version
```
